# Administrative Tasks

Administrative tasks for Data Space Integration that allow you to update connector details when changing landscapes or de-provision the service when switching instances or discontinuing data space usage.

As an administrator for Data Space Integration, you can perform the following actions:

-   [Changing the Landscape of Your Connector](changing-the-landscape-of-your-connector-3a810c8.md): You want to update your connector details because you changed its landscape.

    To perform this action, you need the role collections `PI_Administrator` and `DataspaceTechnicalAdmin`.

-   [Deprovisioning Data Space Integration](deprovisioning-data-space-integration-67c080d.md): You want to de-provision Data Space Integration because you want to use a different SAP Integration Suite instance or you've decided not to work with data spaces anymore.

    To perform this action, you need the role collections `PI_Administrator` and `Integration_Provisioner`.
