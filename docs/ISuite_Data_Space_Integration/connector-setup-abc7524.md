# Connector Setup

Learn about the steps required to create and set up a new connector.

> ### Note:  
> To perform the connector setup on the *Settings* \> *Data Spaces* page, you must be a **technical administrator**. The settings available are defined during the initial setup, updated when necessary, or during the creation of a new connector.
> 
> For information on authorizations and roles, see [Identity and Access Management for Data Space Integration](identity-and-access-management-for-data-space-integration-211c66a.md).

To create and configure a new connector, perform the following steps:

1.  You can **create more than one connector** and switch between connectors on every screen, provided you have the required authorization. See [Creating Connectors](creating-connectors-2c42600.md).
2.  To give users **access to a connector**, you need to create roles for each connector in your subaccount in SAP BTP cockpit. See [Creating Role Collections for Connectors](creating-role-collections-for-connectors-119b70a.md).
3.  Complete the onboarding of any new connector. See [Configuring Connector Setup](configuring-connector-setup-4909d3f.md).
