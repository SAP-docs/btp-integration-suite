# Manage Business Data Graphs using API Composition Configuration API

As a API Composition Key User, you can create a new business data graph using the API Composition Configuration API.

The API Composition Configuration API simplifies the process of managing your business data graphs automatically. Whether you need to create, update, or delete business data graphs, this API provides an automated option to achieve these tasks.

> ### Note:  
> The API Composition Configuration API is currently limited to managing business data graphs. Managing extensions is not supported. Its primary focus is to enable a seamless integration of business data graph management with your existing CI/CD pipelines.
