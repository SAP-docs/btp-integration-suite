# Design Guidelines View for Script Collection

This tab shows the list of design guidelines applicable to the script collection artifact.

You must follow the design guidelines enabled on the tenant that helps you improve the quality of your artifacts. Essential guidelines can't be skipped to help maintain compatibility with future software upgrades.

For more information on how you can validate script collection artifacts, see [Design Guidelines View](design-guidelines-view-d62dfe0.md)
