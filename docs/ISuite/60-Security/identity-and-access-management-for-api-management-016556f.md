# Identity and Access Management for API Management

Here you can find information on how to secure API Management applications using user authentication and authorization.


<table>
<tr>
<th valign="top">

Section

</th>

Description

<td valign="top">

User Authentication

</td>
<td valign="top">

To access API Management application one needs to have a valid SCN user registered with API Management solution. The SCN credentials should be used for logon to API Management solution.

</td>
</tr>
<tr>
<td valign="top">

Authorization

</td>
<td valign="top">

In API Management, you provide authorization to users by assigning relevant roles. For more information on how to provide authorizations, see [Assigning Role Collections to Users](https://help.sap.com/viewer/66d066d903c2473f81ec33acfe2ccdb4/Cloud/en-US/80bb02ed7bf34570b7e8b3ad01aa9ad3.html "Role collections enable you to group together the roles you create. The role collections you define can be assigned to users logged on to SAP ID service.") :arrow_upper_right:.

</td>
</tr>
