# Migrating Agreements

Migrate an agreement that has an outdated format to the latest format.



## Context

> ### Caution:  
> The support for the outdated agreement format ends **by 31 August 2026**. Please migrate to the latest format as soon as possible to avoid maintenance issues.

Agreements now have a latest format that allows them to choose between two different creation modes. These creation modes, copy and reference, enable easy maintenance of agreements and are introduced in alignment with the latest agreement template format. If you have any agreements that were created before this format, you need to migrate them to the latest format to continue editing or copying. To do so, perform the following steps.



## Procedure

1.  Log in to your application and navigate to the *Agreements* tab.

2.  Select the agreement that you want to migrate. Agreements with outdated format have the *Outdated Format* label under their name.

3.  Choose *Migrate*. In the *Confirm Migration* dialog box, choose *Confirm* to migrate the agreement to the latest format.

    You can now start editing the agreement.
