# Exporting a Trading Partner Agreement

Learn how to export a trading partner agreement as a PDF file.



## Context

You can export trading partner agreements as PDF files that you can view offline, share, or archive. The exported file includes all agreement details in a printable format and is automatically saved to your default browser download location.



## Procedure

1.  Log on to SAP Integration Suite and go to *Design* \> *B2B Scenarios*.

2.  Navigate to the *Agreements* tab, which displays a list of the agreements created in the system.

3.  Open the agreement that you want to export.

4.  In the header area of the agreement, choose *Export as PDF*. The file download starts automatically.

    The file name follows the following format, with the suffix being the timestamp of the download: `Agreement Name_YYYYMMDDHHMMSS.pdf`
