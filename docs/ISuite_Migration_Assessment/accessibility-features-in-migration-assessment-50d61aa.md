# Accessibility Features in Migration Assessment

To optimize your experience of Migration Assessment, SAP Integration Suite provides features and settings that help you use the software efficiently.



> ### Note:  
> Migration Assessment is a capability of SAP Integration Suite. For this reason, accessibility features for SAP Integration Suite also apply. See [Accessibility Features in SAP Integration Suite](https://help.sap.com/viewer/51ab953548be4459bfe8539ecaeee98d/CLOUD/en-US/e3e04a8258754b0d818f70324138549f.html "To optimize your experience of SAP Business Technology Platform (SAP BTP), SAP Integration Suite provides features and settings that help you use the software efficiently.") :arrow_upper_right:.

For more information on screen reader support and keyboard shortcuts, see [Accessibility for End Users](https://help.sap.com/docs/SAPUI5/bc5a64aac808463baa95b4230f221716/f562835d0b4e44129aa24a17551a0baa.html).
