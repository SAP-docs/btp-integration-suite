# Accessibility in APIs and Integrations

SAP Integration Suite provides a range of accessibility features designed to ensure an inclusive and efficient user experience across its capabilities, including API Management and Cloud Integration. These features support users with diverse needs by enabling interaction through assistive technologies, keyboard navigation, and optimized interface settings.

Built on SAPUI5, the user interfaces across Integration Suite inherit consistent accessibility capabilities such as screen reader compatibility, semantic controls, and standardized keyboard handling.
