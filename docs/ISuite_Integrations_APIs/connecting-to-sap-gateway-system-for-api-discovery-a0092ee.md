# Connecting to SAP Gateway System for API Discovery

Establish connectivity to your backend systems through SAP Gateway to discover and explore the APIs they expose.

To create an API artifact and access backend systems, first establish a connection through the SAP Gateway service. Once connected, you can discover available APIs and use them to create the artifact.

To enable discovery, you must create a destination using the details of the SAP Gateway system. This destination allows the platform to connect to SAP Gateway and retrieve the list of available backend APIs. Once the destination is configured successfully, you can view and select the APIs exposed by the backend system through SAP Gateway. These APIs can now be used to create an API artifact.
