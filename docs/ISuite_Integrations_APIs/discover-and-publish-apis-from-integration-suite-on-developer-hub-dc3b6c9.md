# Discover and Publish APIs From Integration Suite on Developer Hub

Content administrators can discover APIs from the SAP Integration Suite, organize them into products, and publish these products to the Developer Hub catalog. This makes the APIs available for application developers to view and consume.

Once the products are listed in the catalog, developers can consume them by creating new subscriptions or adding them to existing ones.
