# Analyze APIs with AI Assistance

The Conversational Analytics feature provides AI-powered insights into API usage and performance through natural language interaction. Once enabled, you can use the AI Assistant to analyze API call patterns, response times, and usage trends.



## Context

The API Management capability in SAP Integration Suite includes a generative AI–powered AI Assistant that helps you analyze API usage and performance data.

Use the AI Assistant in the API Analytics dashboard to explore your API data using natural language. Instead of manually configuring filters or building custom queries, you can ask questions and receive AI-generated insights that you can add directly to your analytics view. You can ask questions about:

-   **Traffic patterns:** for example, API call volumes over a specific time range
-   **Performance:** such as, response times or latency trends
-   **Usage trends:**including top-consumed APIs, active applications, or error patterns

The AI Assistant analyzes the relevant analytics data and returns a natural language response, which you can review and optionally add to your dashboard view as a reusable insight.

**Example Prompt:** *"Analyze API traffic for the last 30 days. Break down the top 5 APIs by call volume and include any insights about how they’re being used."*

> ### Caution:  
> SAP does not assume responsibility for the quality of the content provided to or generated by the underlying generative AI models, including but not limited to bias, hallucinations, and inaccuracies. You are responsible for verifying all AI-generated content before relying on it.

## Enabling Conversational Analytics

To enable the Conversational Analytics feature through the SAP Integration Suite, follow the steps below:

1.  Log on to *SAP Integration Suite*.

2.  From the left navigation pane, choose *Settings* \> *Artificial Intelligence*.

3.  Locate *Conversational Analytics* in the features list and toggle *Activation Status* to *ON*.

    **Result:****Conversational Analytics** is active. You can begin analyzing API performance by entering natural language queries in the AI Assistant.




## Launching the AI Assistant from the Analytics Dashboard

The analytics dashboard in API Management provides charts and metrics that help you understand the usage and performance of your APIs. With the AI Assistant, you can query your analytics data conversationally, without manually navigating charts.

To launch the AI Assistant:

1.  Log on to **SAP Integration Suite**.

2.  From the left navigation pane, choose *Analyze*. The analytics dashboard appears.

3.  In the top-right corner, choose *Launch Assistant*.




## Querying Analytics Data

1.  Type your question in the input field using natural language and choose *Generate*.

    Example: *What is the average response time of the ProductCatalog API over the last 30 days?*

    > ### Note:  
    > If you don’t specify a time range in your query, the AI Assistant returns data for the current week by default.
    > 
    > For example, *"Show me the top 5 APIs by traffic volume"* returns top APIs for the current week.

2.  Review the response. The AI Assistant returns results in one of the following formats:

    -   **Chart:** an interactive visual \(bar, line, pie, and others\).

    -   **Table:** tabular data for detailed inspection.

    -   **Text summary:** natural-language analysis with key insights.


    > ### Note:  
    > Some query types in conversational analytics currently support only specific output formats:
    > 
    > -   **Anomaly detection:** Results are displayed only in **tabular** view.
    > -   **Predictions:** Results are displayed only in **chart** view.




## Saving Charts

To save your results, choose *Add to View* option. You can save the charts to an existing custom view or create a new custom view.



### Behavior on Save

-   **Time range alignment:** When you save a chart to a custom view, the chart adopts the time range configured for that custom view.
-   **Display in preview vs. saved view:** If a chart contains a large amount of data, the AI Assistant preview may compress the chart, and not all fields may be visible. Once you save the chart to a custom view, all fields are displayed correctly.

## Known Constraints

-   The AI Assistant provides read-only access to analytics data. It cannot create, modify, or delete APIs or configurations.

-   Queries are limited to the dimensions and measures available in the analytics dashboard.

-   Prediction accuracy depends on the volume of available historical data.

-   AI-generated responses may occasionally require you to clarify or rephrase your query.

-   Real-time data may not be available; results reflect the latest data refresh cycle of the analytics dashboard.

-   Do not include sensitive or personally identifiable information \(PII\) in your prompts.
