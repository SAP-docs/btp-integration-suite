# Process Flow Step

Process steps enable you to structure and organize the execution flow of an API artifact by handling exceptions and dividing complex integration logic into smaller, reusable fragments. You can add these steps to the policy model of your API artifact to improve error handling and process modularity.



## Use

You use process steps in an API artifact to:

-   Capture and handle exceptions that occur during the execution of an integration process.
-   Simplify complex integration scenarios by dividing the main process into smaller, reusable fragments.
-   Combine reusable process fragments to achieve the overall integration logic.
