# Using Git Operations

Git operations enable users to synchronize integration artifacts between Cloud Integration and a Git repository.

The supported Git operations are:

-   *Git Push*: To upload integration artifact changes to the connected Git repository.
-   *Git Pull*: To retrieve the latest changes from the connected remote Git repository into the integration artifact.
