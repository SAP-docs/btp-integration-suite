# Initial Setup of SAP Cloud Integration in the Kyma Environment

In the Kyma environment, connect SAP Cloud Integration to your cluster and manage it using SAP BTP Service Operator.

As a prerequisite, you must have subscribed to the service *Process Integration* and the service instance *Process Integration Runtime* in the same SAP BTP subaccount.

For detailed information about how to connect SAP Cloud Integration to your cluster, refer to [Using SAP BTP Services in the Kyma Environment](https://help.sap.com/docs/btp/sap-business-technology-platform/using-sap-btp-services-in-kyma-environment).
