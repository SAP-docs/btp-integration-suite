# Managing License for Type Systems

> ### Note:  
> This procedure is not relevant for SAP Integration Suite users as the licensing of the type systems is provided along with the software subscription.

You can access Integration Advisor when you purchase an Enterprise Edition license for SAP Cloud Integration. However, you need to purchase licenses for the type systems used in SAP Integration Advisor, like *X12* or *EDIFACT* separately.

For information about purchasing these type system licenses, please raise an ticket under the component *LOD-HCI-PI-SUG*.

Update or verify the license you have purchased on SAP store as follows:

1.  Log on to Integration Advisor and access the *Profile* section.
2.  The *Licenses* tab list the type systems and their license information.

    > ### Note:  
    > To update the license information, choose :arrows_clockwise:.


> ### Note:  
> In case of a subaccount move, check for the type system licenses and raise an incident under the component *LOD-HCI-PI-SUG* to move the licenses to your new subaccount.
