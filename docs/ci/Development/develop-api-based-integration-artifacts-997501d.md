# Develop API-Based Integration Artifacts

Develop OData API-based integration scenarios, deploy, and manage your artifacts in one place.

You can create integration artifacts of types OData APIs. The API-based artifacts generate a default integration design that you can further extend to meet your requirements. After you complete your integration design, you can save and deploy the artifacts. Finally, you can create your own tiles in the monitoring section to view your API-based artifacts that are deployed. For more information, see [SAP Cloud Integration – Step towards Building API Based Integrations](https://blogs.sap.com/2021/01/08/sap-cloud-integration-step-towards-building-api-based-integrations/).
